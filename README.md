## Convention 종류

| 태그이름 | 설명                                    |
| :------- | :-------------------------------------- |
| feat     | 새로운 기능 추가                        |
| fix      | 버그 수정                               |
| design   | css 등 사용자 UI 디자인 수정            |
| refactor | 코드 리팩토링                           |
| comment  | 필요한 주석 추가 및 변경                |
| docs     | 문서 수정                               |
| chore    | 패키지 매니저 설정                      |
| rename   | 파일 혹은 폴더명 수정하거나 옮기는 작업 |
| remove   | 파일을 삭제하는 작업만 하는 경우        |
